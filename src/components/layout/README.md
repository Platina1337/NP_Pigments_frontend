# Мобильный нижний навбар

## Как это работает

- **Десктоп**: Показывается обычный верхний Header с полной навигацией
- **Мобильные устройства (< 768px)**: Верхний Header скрыт, показывается единый нижний навбар

## Структура мобильного навбара (7 элементов в одном ряду):

### 1. **Поиск** - открывает модальное окно поиска
### 2. **Главная** - навигация на главную страницу
### 3. **Каталог** - навигация в каталог товаров
### 4. **Бренды** - навигация по брендам
### 5. **Парфюмы** - навигация к парфюмерии
### 6. **Центральная кнопка корзины** - плавающая с градиентом и анимацией
### 7. **Профиль** - навигация в профиль пользователя
### 8. **Избранное** - навигация к избранным товарам

## Особенности:

### **Центральная кнопка корзины**
- Плавающая кнопка в центре с градиентным фоном
- Вращающаяся анимированная рамка
- Счетчик товаров с bounce анимацией
- Пульсирующий эффект

### **Активные состояния**
- Shimmer эффект для активных элементов
- Неоновое свечение
- Градиентные акценты

### **Модальные окна**
- Полноэкранный поиск с быстрыми категориями
- Боковая корзина (справа)

### **Специальные эффекты**
- Плавные анимации и микро-взаимодействия
- Touch-friendly размеры (минимум 44px)
- Safe-area поддержка для современных устройств

## Технические детали:

- **Один блок навбара** (не два как было раньше)
- CSS классы: `mobile-nav-*` для анимаций
- Responsive: `md:hidden` для показа только на мобильных
- Анимации: CSS keyframes для всех эффектов
